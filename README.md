# LED Brightness Control Using Web Interface

## Project Overview

This project controls LED brightness using Arduino UNO and a web-based interface.

The brightness can be adjusted in real time using a slider on a webpage.

The slider sends data to Arduino through Web Serial API.

Arduino receives the brightness value and controls the LED using PWM.

---

## Technologies Used

- Arduino UNO
- HTML
- CSS

- Web Serial API



## Features

- Real-time brightness control
- PWM LED control
- Browser-to-Arduino communication
- Interactive web interface
